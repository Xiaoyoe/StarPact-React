# 实现自定义程序窗口导航栏标题栏

## 1. 项目分析
- **项目类型**：React + Vite + Tailwind CSS + Electron桌面应用
- **状态管理**：Zustand
- **动画效果**：Framer Motion
- **主题系统**：完整的CSS变量主题系统，支持多种主题

## 2. 实现目标
- 创建自定义的窗口标题栏组件
- 适配现有主题系统
- 实现标准窗口控制功能（最小化、最大化、关闭）
- 支持窗口拖放功能
- 集成到现有应用布局中

## 3. 实现步骤

### 步骤1：创建TitleBar组件
- 创建 `src/components/TitleBar.tsx` 文件
- 实现标题栏布局和样式
- 添加窗口控制按钮
- 集成主题变量

### 步骤2：实现窗口控制功能
- 在 `src/main/ipc/` 目录下添加窗口控制相关的IPC处理
- 实现最小化、最大化、关闭窗口的功能
- 添加窗口状态检测（最大化/还原）

### 步骤3：集成到应用布局
- 修改 `src/App.tsx`，在主内容区域顶部添加TitleBar组件
- 调整布局结构，确保标题栏与侧边栏和内容区域协调
- 确保响应式设计

### 步骤4：实现窗口拖放功能
- 添加Electron的拖放区域设置
- 确保标题栏可拖动
- 避免控制按钮区域触发拖放

### 步骤5：主题适配
- 确保标题栏在所有主题下正常显示
- 测试不同主题的视觉效果
- 调整样式以适配各主题

### 步骤6：测试和优化
- 测试窗口控制功能
- 测试主题切换效果
- 优化拖放性能
- 确保与现有功能无冲突

## 4. 技术要点
- 使用CSS变量实现主题适配
- 利用Electron的IPC通信实现窗口控制
- 使用Framer Motion添加平滑过渡效果
- 确保拖放区域的正确设置
- 保持与现有设计风格的一致性

## 5. 预期效果
- 美观的自定义标题栏，与应用整体风格一致
- 完整的窗口控制功能
- 支持所有现有主题
- 流畅的拖放体验
- 响应式设计，适配不同窗口尺寸
